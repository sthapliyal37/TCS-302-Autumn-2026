## Course Outcomes

After successfully completing the course, students will be able to:

- **CO1:** Describe fundamental data structures and analyze how the choice of data structures affects program efficiency.
- **CO2:** Compare different data structures in terms of time complexity, space complexity, and practical applications.
- **CO3:** Identify and select suitable data structures for solving real-world computational problems.
- **CO4:** Implement searching, insertion, deletion, traversal, and other operations on linear and non-linear data structures.
- **CO5:** Apply advanced data structures including Binary Search Trees, AVL Trees, Hash Tables, Priority Queues, and Abstract Data Types.
- **CO6:** Design efficient solutions by combining multiple data structures and algorithms for innovative problem-solving.

## C Program Compilation

Compile a C program using GCC:

```bash
gcc program.c -o program
```

Run the executable:

### Linux or macOS

```bash
./program
```

### Windows

```bash
program.exe
```

Compile with recommended warnings:

```bash
gcc -Wall -Wextra -pedantic program.c -o program
```

---

## Coding Standards

Students should follow these guidelines:

1. Use meaningful variable and function names.
2. Divide large programs into functions.
3. Validate user input.
4. Check the result of dynamic memory allocation.
5. Free dynamically allocated memory.
6. Add comments only where necessary.
7. Mention the time and space complexity of each operation.
8. Test programs using normal, boundary, and invalid inputs.

---

## Suggested Learning Process

For every topic:

1. Read the concept notes.
2. Perform a manual dry run.
3. Study the algorithm or pseudocode.
4. Implement the algorithm independently.
5. Analyse time and space complexity.
6. Test edge cases.
7. Solve the assigned practice problems.
8. Compare the solution with alternative approaches.
